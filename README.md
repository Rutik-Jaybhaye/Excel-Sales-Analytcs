# Excel-Sales-Analytics

## 📊 Project Overview
This project focuses on analyzing sales data using Microsoft Excel. The goal is to generate meaningful insights from raw sales data and present them in a professional and visually appealing format through various reports.

## 📁 Files Included

- **FianancialReport.pdf** – A detailed financial analysis of sales and expenses.
- **IndiaSalesReport.pdf** – Region-specific sales analysis for India.
- **Second_report.pdf** – Additional insights or revised metrics based on updated data.
- **UpdatedP&L.pdf** – Final version of the Profit and Loss statement, updated with latest corrections.

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Cleaning Techniques
- Conditional Formatting
- Basic Excel Formulas (SUMIF, VLOOKUP, IF, etc.)

## ✅ Key Insights

- Identified top-performing regions and products.
- Tracked profit margins and financial performance.
- Highlighted areas for improvement in operational efficiency.
